## 🚨 Important Notes

### Critical Information
- ⚠️ ProjectId is **required** for every sample
- ⚠️ Cascade delete is **enabled** (deleting project deletes samples)
- ⚠️ All timestamps are in **UTC format**
- ⚠️ Only **Title** field is required

### Before You Start
✅ Database must be migrated
✅ Application must be running
✅ Project must exist before creating samples
✅ Use UTC timestamps

### Common Mistakes
❌ Creating sample without valid ProjectId
❌ Not migrating database before running
❌ Using local time instead of UTC
❌ Expecting UpdatedAt to be set on new samples
